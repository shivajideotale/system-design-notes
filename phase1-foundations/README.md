# Phase 1: Foundations & Refresher (Weeks 1–4)

This phase covers the foundational concepts necessary for advanced system design, including networking, operating systems, data structures, and Java platform internals. These topics provide the groundwork for understanding scalable and distributed systems.

## Topics

- [Networking Fundamentals](networking-fundamentals.md)
- [Operating System Concepts](operating-system-concepts.md)
- [Data Structures & Algorithms](data-structures-algorithms.md)
- [Java Platform Deep Dive](java-platform-deep-dive.md)

## Learning Objectives

By the end of this phase, you should be able to:
- Explain core networking protocols and their role in system design
- Understand OS-level abstractions for processes, memory, and I/O
- Apply advanced data structures for system design problems
- Dive deep into JVM internals and modern Java features
