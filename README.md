# System Design Roadmap Learning Notes

This repository contains my personal learning notes for the 12-month system design roadmap, organized by phases and topics. The notes are designed to help me master system design concepts at a senior level, targeting Staff/Principal Engineer roles.

The content is structured as a navigable website published via GitHub Pages for easy access from anywhere.

## Roadmap Overview

Based on the [SYSTEM_DESIGN_ROADMAP.md](SYSTEM_DESIGN_ROADMAP.md), this is a 52-week journey covering 10 phases of system design topics.

- [Phase 4: Database Design & Storage](phase4-database-design/) (Weeks 17–20)

- [Phase 6: Microservices & API Design](phase6-microservices/) (Weeks 25–28)
